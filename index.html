Pet Store Database

This project was my first dynamic web application, developed as a class assignment at Florida State Univeristy. I was put up with the task of creating a method of organizing and communicating sales information for an imaginary pet store. I created a mysql database to store large amounts of metadata about the many pet stores owned by the imaginary corporate giant owning stores all across the world. In order to help all users interface with this database, I created a mobile-friendly user interface using HTML, CSS, and a variety of other front end technologies. PHP was utilized to connect the database and the web interface together. The final product allows users to add new pet stores to the database, edit existing records, as well as delete them from the database.

Drake Simulator

Hackathons always bring about the craziest and most creative ideas. Drake Simulator is one such project that came out of HackFSU 2016. Have a seat, because this one is ridiculous. This web game was built around the story that Drake the rapper is in a very negative mood. Upon visiting the desktop website, you can see just how much he is frowning. It is up to you to make Drake happy again. There is only one way to do this - by utlizing the hit dance craze of 2016, The Dab. By visiting the mobile companion website on your smartphone, each time you dab, Drake gets happier and happer, and you complete your mission. As fun and enjoyable as this project was, it was also very challenging. The correct coordinates of the dance move needed to be calculated, and finding a creative way to integrate the desktop and mobile websites also proved to be difficult. Solving these challenges helped me to understand how to integrate the many web technologies that are avaiable, and put them into a user friendly and enjoyable product.
